# 📖 Miniguia de Estudos — Glossário Inteligente do Universo Tech

## 📌 Introdução

Este miniguia foi elaborado para ajudar iniciantes a compreender os principais conceitos da área de tecnologia de forma simples, didática e prática.

O conteúdo foi desenvolvido com apoio do NotebookLM, utilizando diferentes fontes sobre computação em nuvem, inteligência artificial, Git/GitHub, lógica de programação e metodologias ágeis.

# 🧠 1. Resumos Estruturados por Tema

## ☁️ A. Computação em Nuvem (Cloud Computing)

Cloud Computing é a entrega de serviços como servidores, bancos de dados, armazenamento e softwares através da internet, eliminando a necessidade de manter infraestrutura física local.

### ✅ Principais vantagens
- Redução de custos;
- Escalabilidade;
- Flexibilidade;
- Segurança;
- Acesso remoto.

### 📦 Modelos de Serviço

#### IaaS (Infrastructure as a Service)
O usuário aluga a infraestrutura necessária, como servidores e armazenamento.

**Exemplo:** semelhante a possuir um DVD player e escolher os filmes separadamente.

#### PaaS (Platform as a Service)
Ambiente pronto para desenvolvimento e execução de aplicações.

**Exemplo:** como utilizar uma TV com sistema já preparado para rodar conteúdos.

#### SaaS (Software as a Service)
Software pronto acessado diretamente pela internet.

**Exemplo:** Netflix, Google Drive e Spotify.

---

## 🤖 B. Inteligência Artificial (IA)

A Inteligência Artificial busca desenvolver sistemas capazes de simular habilidades humanas, como interpretação, aprendizado e tomada de decisão.

Atualmente, destaca-se a IA Generativa, responsável pela criação de:
- textos;
- imagens;
- vídeos;
- músicas;
- códigos.

### 📈 Camadas de Oportunidade

#### Usuário de produtividade
Utiliza ferramentas de IA para acelerar tarefas e aumentar eficiência.

#### Builder
Cria automações, agentes inteligentes e soluções utilizando IA para resolver problemas reais.

---

## 🔄 C. Git e GitHub

Git e GitHub são ferramentas fundamentais para colaboração e versionamento de projetos.

### Git
Ferramenta de controle de versão instalada no computador.

Permite:
- registrar alterações;
- recuperar versões anteriores;
- acompanhar modificações;
- trabalhar em equipe.

Pode ser comparado a uma "máquina do tempo" para projetos.

### GitHub
Plataforma online utilizada para armazenar, compartilhar e colaborar em projetos utilizando Git.

Funciona como um espaço central onde equipes unem e organizam seus códigos.

---

## 💻 D. Lógica de Programação

A lógica de programação é a base do desenvolvimento de software.

Ela permite transformar raciocínios em instruções compreensíveis para computadores.

### 🔹 Conceitos Fundamentais

#### Algoritmos
Sequência lógica de passos para resolver um problema.

**Exemplo:** receita de bolo.

#### Variáveis
Espaços reservados para armazenar informações que podem mudar ao longo da execução.

#### Condicionais
Estruturas que controlam decisões no programa.

**Exemplo:**  
"Se a pessoa for maior de idade, então pode acessar."

---

## ⚡ E. Metodologias Ágeis

Metodologias ágeis são práticas utilizadas para acelerar projetos e melhorar a eficiência das equipes através de ciclos contínuos de melhoria.

### Scrum
Framework baseado em ciclos curtos chamados Sprints, normalmente entre 2 e 4 semanas.

O objetivo é entregar valor rapidamente ao cliente.

### Kanban
Método visual de organização de tarefas utilizando colunas como:
- A Fazer;
- Fazendo;
- Feito.

Ajuda a identificar gargalos e organizar fluxos de trabalho.

# 📚 2. Glossário de Conceitos-Chave

| Conceito | Definição |
|---|---|
| API | Forma de conectar e trocar informações entre sistemas |
| Backlog | Lista de demandas organizadas por prioridade |
| Branch | Linha separada de desenvolvimento para testes e alterações |
| Commit | Registro permanente de alterações no Git |
| Deploy | Processo de publicação de uma aplicação |
| LLM | Grandes modelos de linguagem utilizados em IAs conversacionais |
| MVP | Versão mais simples de um produto funcional |
| Push | Envio das alterações locais para o GitHub |
| Staging Area | Área temporária para revisar alterações antes do commit |
| VCN | Rede virtual privada criada dentro da nuvem |
