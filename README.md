# 📘 Glossário Inteligente do Universo Tech

## 📌 Contexto e Objetivos

Este projeto foi desenvolvido utilizando o NotebookLM, ferramenta de inteligência artificial do Google voltada para organização, síntese e exploração de conteúdos.

O tema escolhido foi a introdução ao universo da tecnologia e desenvolvimento de software, com foco em compreender conceitos frequentemente utilizados na área tech, como Lógica de programação, GitHub, Cloud Computing e metodologias ágeis.

Os principais objetivos deste estudo foram:

- Criar um material de apoio para iniciantes em tecnologia;
- Aprender conceitos técnicos de forma simplificada;
- Explorar estratégias de engenharia de prompts no NotebookLM;
- Desenvolver um glossário reutilizável para estudos futuros;
- Entender como a IA pode auxiliar na organização e interpretação de conteúdos técnicos.

# 🧠 Engenharia de Prompts e Aprendizados

## 🔹 Prompt Inicial

> “Explique conceitos da área de tecnologia de forma didática e acessível para alguém em transição de carreira, usando exemplos práticos e linguagem clara.”

### Resultado

As respostas ficaram mais compreensíveis e menos técnicas, facilitando a assimilação dos conceitos.

---

## 🔹 Prompt para Analogias

> “Explique o conceito de pipeline usando uma analogia do cotidiano.”

### Resultado

A IA utilizou exemplos simples, como restaurantes e pedidos, tornando o conceito mais intuitivo.

---

## 🔹 Prompt para Organização de Estudos

> “Monte uma trilha de aprendizado para iniciantes em tecnologia, organizando os conceitos do mais básico ao mais avançado.”

### Resultado

O NotebookLM conseguiu estruturar uma sequência lógica de aprendizagem.

---

## ⚠️ Dificuldades Encontradas

- Algumas respostas ficaram técnicas demais quando o prompt era muito genérico;
- Certos conceitos apareceram sem contexto prático;
- Foi necessário refinar os prompts para obter respostas mais didáticas;

---

# 💡 Prompts Reutilizáveis

- “Explique este conceito como para um iniciante.”
- “Use uma analogia simples para explicar este termo.”
- “Quais são as aplicações práticas desse conceito?”
- “Explique a diferença entre X e Y.”
- “Monte um resumo estruturado sobre este tema.”
- “Quais conceitos preciso aprender antes deste?”
- “Como esse conceito aparece no mercado de tecnologia?”

---

# 🚀 Considerações Finais

O NotebookLM demonstrou grande potencial como ferramenta de apoio aos estudos, especialmente para organização de conteúdos técnicos e criação de materiais personalizados de aprendizagem.

Além de facilitar a compreensão de conceitos complexos, a ferramenta também auxiliou no desenvolvimento de habilidades relacionadas à engenharia de prompts e pensamento crítico sobre respostas geradas por IA.
