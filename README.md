# 📘 Glossário Inteligente do Universo Tech

## 📌 Contexto e Objetivos

Este projeto foi desenvolvido utilizando o NotebookLM, ferramenta de inteligência artificial do Google voltada para organização, síntese e exploração de conteúdos.

O tema escolhido foi a introdução ao universo da tecnologia e desenvolvimento de software, com foco em compreender conceitos frequentemente utilizados na área tech, como Lógica de programação, GitHub, Cloud Computing e metodologias ágeis.

Os principais objetivos deste estudo foram:

- Criar um material de apoio para iniciantes em tecnologia;
- Aprender conceitos técnicos de forma simplificada;
- Explorar estratégias de engenharia de prompts no NotebookLM;
- Desenvolver um glossário reutilizável para estudos futuros;
- Entender como a IA pode auxiliar na organização e interpretação de conteúdos técnicos.
